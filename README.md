# AnonUserGuy.github.io
A personal website that hosts various utilities including a *Terraria* minimap file visualizer/converter, *Minecraft* world event timeline generator, and *Rhythm Heaven* bytecode ROM offset to RAM offset converter.

https://anonuserguy.github.io/

## Building 
### Prerequisites
- Node.js (v18+ recommended)
- npm

### Install dependencies
```bash
npm install
```
### Build typescript files
```bash
npm run build
``` 
Output website will be generated at ``./dist/``.
